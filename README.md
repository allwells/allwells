## What I Build

**Real-time systems**

WebSocket-powered dashboards and live data interfaces handling high volumes of concurrent users.

**Multi-tenant architectures**

Role-based applications serving multiple user types from a unified codebase, with isolated data, access controls, and protected views.

**SaaS product interfaces**

Feature-rich dashboards covering bookings, wallets, analytics, onboarding, settings, and everything in between.

**Marketplace and platform frontends**

End-to-end frontend systems for two-sided platforms with separate user, client, and admin applications sharing a unified design system.

**CMS architecture**

Sanity and Storyblok implementations with custom schemas, portable text configurations, and editor-friendly content structures.
